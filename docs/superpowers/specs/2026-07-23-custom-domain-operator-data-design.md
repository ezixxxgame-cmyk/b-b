# Домен и реквизиты оператора

## Цель

Сделать `https://www.barberandbarbie.ru` единственным SEO-каноническим адресом сайта и заменить юридические плейсхолдеры проверенными реквизитами оператора.

## Решение

- Главный URL: `https://www.barberandbarbie.ru`.
- Вариант без `www` перенаправляется Vercel на главный URL кодом 308.
- `NEXT_PUBLIC_SITE_URL` имеет приоритет; fallback в коде меняется на главный URL, чтобы canonical, sitemap, robots и JSON-LD не вернулись к адресу `vercel.app`.
- Фавикон остаётся относительным (`/favicon.ico`, `/favicon.svg`), поэтому доступен с нового домена без переноса файла.

- В футере указываются реквизиты оператора без публикации лишних персональных данных.

## Проверка

- Тесты подтверждают новый основной URL и отсутствие плейсхолдеров в юридических документах.
- TypeScript, тесты и production-сборка проходят.
- После deploy: `https://barberandbarbie.ru` перенаправляет на `https://www.barberandbarbie.ru`; canonical, sitemap и JSON-LD используют `www`.
