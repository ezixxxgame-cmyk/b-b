# «Барбер и Барби» — утверждённая дизайн-спецификация

## Направление

Концепция B: камерный ретро-плакат с тату-эстетикой, адаптированный под современную запись с телефона. Визуальная опора — чёрное поле, тёплая бумажная типографика, оранжевый CTA и тонкая декоративная рамка. Хиро повторяет композицию утверждённого превью: фирменная иллюстрация, крупное название, короткое описание, две кнопки и подтверждённые локальные факты.

## Токены

- `ink`: #11100D — основной фон и текст на светлых карточках.
- `paper`: #F2E8D0 — тёплый фон, заголовки и рамки.
- `orange`: #D85B1B — основное действие и детали.
- `burgundy`: #76251C — глубина, активные состояния, подложки.
- `muted`: #B9AA90 — вторичный текст на тёмном фоне.

Типографика: выразительный serif-display для плакатных заголовков, нейтральный sans-serif для интерфейса и компактный utility-стиль для фактов/меток. В коде используются системные fallbacks, чтобы не блокировать первый рендер.

## Композиция

```text
┌─ бумажная страница ───────────────────────────────────────┐
│  ┌─ чёрная рамка ───────────────────────────────────────┐  │
│  │ БАРБЕР И БАРБИ                         ☰              │  │
│  │             [фирменная иллюстрация]                  │  │
│  │                    БАРБЕР И БАРБИ                    │  │
│  │  Стрижки, окрашивание и маникюр в Салавате            │  │
│  │     [Записаться]             [Позвонить]              │  │
│  │   ★ 4,8 на Яндекс.Картах   ⌖ б-р Космонавтов, 13      │  │
│  └───────────────────────────────────────────────────────┘  │
│  интерьер → услуги → преимущества → отзывы → форма → NAP   │
└───────────────────────────────────────────────────────────┘
```

На узких экранах хиро сохраняет порядок превью B, а кнопка записи остаётся доступной через sticky-панель. Реальные фотографии интерьера и работ используются после хиро: так сохраняется утверждённый плакатный вход и появляется доказательство пространства/результата.

## Контентные правила

- Использовать только факты из карточки Яндекс.Карт и подтверждённые пользователем изображения.
- Цены не опубликованы в карточке: показывать честную пометку «уточняется при записи».
- Отзывы — короткие точные цитаты, ссылка на карточку Яндекс.Карт.
- Не добавлять VK, Telegram, акции, выдуманные услуги, мастеров или юридические реквизиты.

## Доступность и движение

Контраст paper/ink и orange/ink проверяется визуально; все интерактивные элементы имеют видимый focus-ring, порядок tab соответствует сценарию записи. Анимация ограничена лёгким появлением рамки и отключается через `prefers-reduced-motion`.
