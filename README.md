# Барбер и Барби

Одностраничный сайт салона в Салавате на Next.js, TypeScript и App Router.

## Локальный запуск

```bash
npm install
npm run dev
```

## Переменные Vercel

В настройках проекта Vercel добавьте только серверные переменные:

- `TELEGRAM_BOT_TOKEN` — токен Telegram-бота.
- `TELEGRAM_CHAT_ID` — chat ID, куда должны приходить заявки.

Секреты не добавляются в репозиторий и не попадают в браузер. Для локальной проверки создайте `.env.local` и удалите его из публикации.

## Что нужно заполнить до публикации

1. В `app/privacy/page.tsx` и `app/consent/page.tsx`: ФИО самозанятого, ИНН, адрес, контакты оператора и дату актуализации.
2. В `app/privacy/page.tsx`: подтвердить сроки хранения и схему передачи данных с юристом.
3. В `app/layout.tsx`: заменить `metadataBase` на фактический домен Vercel/сайта.
4. При необходимости заменить `public/favicon.svg` и добавить утверждённый логотип в `public/images`.

Политика и согласие — рабочие шаблоны, а не заявление о полном юридическом соответствии. Их нужно проверить с юристом после внесения реквизитов оператора.
